### 문제 설명

<p>두 개의 자연수를 입력받아 최대 공약수와 최소 공배수를 출력하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에는 두 개의 자연수가 주어진다. 이 둘은 10,000이하의 자연수이며 사이에 한 칸의 공백이 주어진다.</p>

### 출력 

 <p>첫째 줄에는 입력으로 주어진 두 수의 최대공약수를, 둘째 줄에는 입력으로 주어진 두 수의 최소 공배수를 출력한다.</p>

### Idea

#### 💡 유클리드 호제법: 최대공약수 구하기  

- 유클리드 호제법: 두 정수의 __"최대공약수"__ 를 빠르게 찾는 방법의 일종.  

``` text  
1. 큰 수(a)를 작은 수(b)로 나눈다 -> 나머지(c) 저장  
2. 나누는 수(b)를 나머지(c)로 나눈다  
3. 나머지(c) 가 0이 되면 그때의 나누는 수(b) 가 최대공약수이다  
(반복)  
```
*코드에서는 `a`, `b` 의 대소를 비교하지 않았다. 이렇게 해도 문제가 되지 않는 이유는,  
재귀함수를 호출하며 `a`를 `b`로, `b`를 `c`로 대체하는 과정에서 자동으로 `a>b` 인 상황이 되기 때문이다.
</br>

#### 💡 최소공배수 구하기  

최소공배수 구하기는 간단하다. 아래와 같은 식으로 구할 수 있다.  

`(두 정수 a, b 의 최소공배수) = (a * b) / (a, b 의 최대공약수)`
