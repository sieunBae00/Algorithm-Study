### 문제 설명

<p>그래프를 DFS로 탐색한 결과와 BFS로 탐색한 결과를 출력하는 프로그램을 작성하시오. 단, 방문할 수 있는 정점이 여러 개인 경우에는 정점 번호가 작은 것을 먼저 방문하고, 더 이상 방문할 수 있는 점이 없는 경우 종료한다. 정점 번호는 1번부터 N번까지이다.</p>

### 입력 

 <p>첫째 줄에 정점의 개수 N(1 ≤ N ≤ 1,000), 간선의 개수 M(1 ≤ M ≤ 10,000), 탐색을 시작할 정점의 번호 V가 주어진다. 다음 M개의 줄에는 간선이 연결하는 두 정점의 번호가 주어진다. 어떤 두 정점 사이에 여러 개의 간선이 있을 수 있다. 입력으로 주어지는 간선은 양방향이다.</p>

### 출력 

 <p>첫째 줄에 DFS를 수행한 결과를, 그 다음 줄에는 BFS를 수행한 결과를 출력한다. V부터 방문된 점을 순서대로 출력하면 된다.</p>

---

### Idea  

#### 처음 생각

처음에는 '정점 번호가 작은 것을 먼저 방문한다' 라는 조건에 `priority_queue` 를 쓰려고 했다.   
그러나 구현 과정에서 `priority_queue` 를 쓰면 일반 `vector` 와 같이 구현할 수 없다는 것을 깨닫고(`pop()` 등 필요해서 비효율), 대신 2차원 벡터의 각 열을 `sort()` 시키기로 했다.  

---

#### 구현  

그동안 DFS/BFS 를 쓸 일이 있다면 항상 DFS() 로 구현을 해 왔다. (그게 간단하니까)  
덕분에 BFS 구현에서 살짝 헤멨는데, 그래도 코드를 혼자 힘으로 고칠 수 있었다[Silver II] DFS와 BFS - 1260
