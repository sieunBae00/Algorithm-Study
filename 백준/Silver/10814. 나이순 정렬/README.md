### 문제 설명

<p>온라인 저지에 가입한 사람들의 나이와 이름이 가입한 순서대로 주어진다. 이때, 회원들을 나이가 증가하는 순으로, 나이가 같으면 먼저 가입한 사람이 앞에 오는 순서로 정렬하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 온라인 저지 회원의 수 N이 주어진다. (1 ≤ N ≤ 100,000)</p>

<p>둘째 줄부터 N개의 줄에는 각 회원의 나이와 이름이 공백으로 구분되어 주어진다. 나이는 1보다 크거나 같으며, 200보다 작거나 같은 정수이고, 이름은 알파벳 대소문자로 이루어져 있고, 길이가 100보다 작거나 같은 문자열이다. 입력은 가입한 순서로 주어진다.</p>

### 출력 

 <p>첫째 줄부터 총 N개의 줄에 걸쳐 온라인 저지 회원을 나이 순, 나이가 같으면 가입한 순으로 한 줄에 한 명씩 나이와 이름을 공백으로 구분해 출력한다.</p>

### Idea  

#### 처음 생각  

처음에는 `sort()` 함수의 세 번째 인자로 커스텀 비교 함수를 넣어서, 그 함수에서  
``` text
1. 나이 비교
2. 나이가 같다면 원본 배열에서 찾아, 인덱스를 비교
(이를 위해 배열의 복사본을 준비해 두었다)
```
를 수행하려고 했다. 코드는 아래와 같다.  
``` cpp
bool sort_age(info a, info b) { // 커스텀 비교 함수: a가 b보다 앞에 와야 하면 True 를 반환
	if (a.first < b.first) return true; // 1. 나이 비교 -> 어린 쪽이 앞으로
	else if (a.first == b.first) { // 나이가 같다면
		int a_index = find(V.begin(), V.end(), a) - V.begin();
		int b_index = find(V.begin(), V.end(), b) - V.begin();
		if (a_index < b_index) return true; // 2. 가입순서 비교 -> 원본 배열에서 인덱스가 작은 쪽이 앞으로
		else return false;
	}
	else return false; // 나이가 많다면 뒤로
}
```
🛑 그러나 시간 초과로 실패했다. `find()` 함수가 시간을 너무 많이 잡아먹기 때문.  
찾아보니 `sort()` 함수 외에도 `stable_sort()` 함수가 있다는 사실을 알게 되었다.  
</br>

- 💡 __`stable_sort()`__: 커스텀 비교 함수에서 비교 값이 동일하다면 "__원래의 순서를 유지__ 한다." !!  
  즉, 처음 시도했던 커스텀 함수에서의 "나이가 같을 경우"에 실행되는 코드를 대체할 수 있는 것이다.  
  
`stable_sort()` 를 사용하니 시간 초과 없이 통과할 수 있었다. (배열의 복사본도 만들 필요 없이 원본 배열 하나에서 정렬하면 된다)
