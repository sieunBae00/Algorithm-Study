### 문제 설명

<p>알파벳 소문자로 이루어진 N개의 단어가 들어오면 아래와 같은 조건에 따라 정렬하는 프로그램을 작성하시오.</p>

<ol>
	<li>길이가 짧은 것부터</li>
	<li>길이가 같으면 사전 순으로</li>
</ol>

<p>단, 중복된 단어는 하나만 남기고 제거해야 한다.</p>

### 입력 

 <p>첫째 줄에 단어의 개수 N이 주어진다. (1 ≤ N ≤ 20,000) 둘째 줄부터 N개의 줄에 걸쳐 알파벳 소문자로 이루어진 단어가 한 줄에 하나씩 주어진다. 주어지는 문자열의 길이는 50을 넘지 않는다.</p>

### 출력 

 <p>조건에 따라 정렬하여 단어들을 출력한다.</p>

### Idea

#### 처음 생각

sort() 함수를 쓰면 기본적으로 사전 순으로는 정렬이 가능하다.  

- 🛑 그런데 '길이가 짧은 것부터' 라는 문제 조건은 어떻게 구현할 것인가?  

⭐️ sort() 함수의 **세 번째 인자** 로 직접 만든 '비교 함수'를 넣을 수 있다.  

> '길이가 짧은 것부터 & 길이가 같으면 사전 순으로' 를 판단하는 함수를 하나 만든다.  
> (bool 자료형 반환, a 가 b 보다 앞에 와야 하는 경우 true)   

*이때, 비교 함수의 인자를 따로 넣어 주지 않아도 된다.  
(sort() 함수가 생각 이상으로 똑똑해서, 알아서 벡터의 원소 두 개씩을 비교 함수의 인자로 넣어 비교해 준다.)   
</br>

- 🛑 '중복된 단어는 하나만 남기고 제거한다' 라는 조건은 어떻게 구현하나?  

> 💡 `unique()` + `erase()` 함수를 통해 중복을 제거한다.  

- `unique()`: 범위(range) 를 인수로 받아, "연속된" 중복되는 원소를 배열의 맨 뒤로 보낸다.  
  '맨 뒤로 이동시킨 원소들의 시작 주소를 반환한다. (*전체 배열 사이즈는 변하지 않는다. !!)  
- `erase()`: 범위(range) 를 인수로 받아, 해당 범위의 원소를 삭제한다.  

따라서 다음과 같이 unique() 와 erase() 함수를 함께 사용하므로써, 원소의 중복을 제거할 수 있다.  
``` cpp
erase(unique(V.begin(), V.end()), V.end());
```
