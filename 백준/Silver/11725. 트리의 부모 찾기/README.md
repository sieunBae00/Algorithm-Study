### 문제 설명

<p>루트 없는 트리가 주어진다. 이때, 트리의 루트를 1이라고 정했을 때, 각 노드의 부모를 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 노드의 개수 N (2 ≤ N ≤ 100,000)이 주어진다. 둘째 줄부터 N-1개의 줄에 트리 상에서 연결된 두 정점이 주어진다.</p>

### 출력 

 <p>첫째 줄부터 N-1개의 줄에 각 노드의 부모 노드 번호를 2번 노드부터 순서대로 출력한다.</p>

---

### Idea

#### 정점의 연결 표현하기
- 자식 노드가 몇 개일지 모른다:
> 💡 2차원 배열을 선언하고, `push_back()`으로 저장.  

- ⚠️ 연결된 두 정점은 누가 부모인지 알 수 없으므로, __양쪽에__ 저장한다.  

#### 부모 노드를 저장하는 배열
- 부모 노드의 번호를 2번 노드부터 차례로 출력해야 한다:  
노드의 방문 순서를 알 수 없으므로, 그때그때 출력 불가. 따라서 parent배열을 만들어 부모 노드의 번호를 저장한 후 출력한다.

#### DFS 함수
- 루트가 1이므로, 1을 집어올리고 그 밑에 꿰어진 노드들을 확인한다.
> tree 배열에 저장된 노드를 타고 내려가며 확인한다: 💡 DFS! (recursion)  


- ⚠️ 부모를 확정하여 parent배열에 저장하는 위치에 주의한다.  
> 💡 __부모가 항상 먼저 방문된다__: visited배열이 아직 false이면 자식 노드이다.
