### 문제 설명

<p>준규는 숫자 카드 N장을 가지고 있다. 숫자 카드에는 정수가 하나 적혀있는데, 적혀있는 수는 -2<sup>62</sup>보다 크거나 같고, 2<sup>62</sup>보다 작거나 같다.</p>

<p>준규가 가지고 있는 카드가 주어졌을 때, 가장 많이 가지고 있는 정수를 구하는 프로그램을 작성하시오. 만약, 가장 많이 가지고 있는 정수가 여러 가지라면, 작은 것을 출력한다.</p>

### 입력 

 <p>첫째 줄에 준규가 가지고 있는 숫자 카드의 개수 N (1 ≤ N ≤ 100,000)이 주어진다. 둘째 줄부터 N개 줄에는 숫자 카드에 적혀있는 정수가 주어진다.</p>

### 출력 

 <p>첫째 줄에 준규가 가장 많이 가지고 있는 정수를 출력한다.</p>

### Idea

#### 처음 생각

처음에는 우선순위 큐를 활용하는 방식을 떠올렸지만, 조금 더 생각해 보니 그냥 배열에 저장&정렬 후 순회하며 갯수를 세는 쪽이 맞다는 생각이 들었다.  
</br>

---

#### 구현

인덱스 2개(`curr`, `next`) 를 두고, 현재 정수와 다음 정수를 비교하여 같으면 갯수(`cnt`) 를 +1, 다르면 갯수(`cnt`) 를 초기화한다.  
현재 갯수가 최고 기록(`max_cnt`) 을 갱신하면 최고 기록과 카드 번호(`ans`) 를 업데이트한다.   
(*최고 갯수인 카드가 여러 개라면 작은 것을 출력하므로, 갯수가 같은 경우는 원래의 카드 번호를 유지한다.)  
</br>

- ⚠️ 처음 구현 시 '다른 숫자가 등장할 경우' 에만 '기록 갱신 여부' 를 판단하도록 하여 문제가 있었다.  
  다음과 같이 기록 갱신 코드를 '`else`문 내'에 적는다면, 배열의 마지막에 다른 숫자가 등장하지 않고 끝날 경우 기록 갱신 코드를 거치지 못하게 되어,  
  업데이트가 이루어지지 못할 수 있다.  

``` cpp
while (next <= num.size() - 1) { 
    if (num[curr] == num[next]) cnt++; // 같은 숫자라면 갯수 +1
    else {cnt = 1; // 다른 숫자라면 갯수 초기화
        // ❌ 기록 갱신 여부 코드를 else 문 안에 구현함 !!
        if (cnt > max_cnt) { 
            max_cnt = cnt; 
            ans = num[curr]; 
        }
    }
    // 인덱스 한 칸씩 이동
    curr++;
    next++;
}
```

기록 갱신 여부를 판단하는 코드를 `else`문 밖으로 빼내어 항상 판단하도록 함으로써 이를 해결할 수 있다.
