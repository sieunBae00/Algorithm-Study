### 문제 설명

<p>배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.</p>

### 입력 

 <p>첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.</p>

### 출력 

 <p>첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.</p>

---

### Idea

`N`의 최대 자리수는 10이므로, 시간에 제약이 없을 것 같아 `pirority_queue<>` 에 각 자리 숫자를 넣고, 하나씩 `pop()` 하도록 구현하였다.  
</br>

- `String(문자열)` 에서 `char(문자)` 를 하나씩 떼어내기  
``` cpp
string S;  
for(char c : S){...}  
```
위와 같이 쓸 수 있다. (벡터 등에서와 동일)
